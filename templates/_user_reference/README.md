## Использование

Эти файлы — источник для разметки координатных карт `fields.json` и для pixel-diff тестов:

```python
def test_pixel_perfect_kontur_reference():
    # Берём данные из таблицы выше как input.json
    req = load_reference_input("kontur")
    rendered = pipeline(req).result_pdf
    assert pixel_diff(rendered, "reference_kontur_4pages.pdf") < 10
```

## Замечания

1. `reference_kontur_4pages.pdf` — размер 2.5 MB. Внутри есть растровые слои (вероятно, сканы подписи/печати оператора). Pixel-diff может требовать tolerance больше 10 px на этих областях.
2. `reference_tensor_6pages.pdf` — размер 313 KB, чистый векторный PDF.
3. Для корректного pixel-diff нужно привести оба PDF к одному DPI (рекомендация: 150 DPI).

## Юридическое замечание

Файлы содержат реальные персональные данные (ИНН, ФИО). При публикации репо обязательно:
- Исключить `templates/_user_reference/` из Git (добавлено в `.gitignore`)
- Для публичных тестов использовать синтетические данные
