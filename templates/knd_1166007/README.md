## Отличия от 1166002

Форма 1166007 — более короткая подтверждающая квитанция. Содержит:
- Шапка (представитель/декларант)
- Заголовок "Извещение о вводе сведений..."
- Подтверждающий текст с декларантом, налоговым органом, реквизитами декларации
- Отрицание ошибок: "не содержится ошибок (противоречий)"
- Полное наименование налогового органа + код в конце

В эталоне ТЕНЗОРа **текст ИНН декларанта и название ИФНС напечатаны ЗАГЛАВНЫМИ буквами**
(в отличие от 1166002 — смешанный регистр). Это учитывается при рендере.

## Поля в fields.json

15 динамических:
- `representative_fio_line1` + `representative_inn` — реквизиты справа вверху
- `ifns_code_header` — код ИФНС в тексте-вводной
- `declarant_fio_and_inn_line` + `declarant_inn_explicit` — ФИО + ИНН декларанта
- `declaration_name_knd` — название декларации + КНД
- `correction_and_period`, `correction_number_only`, `tax_period_year_only` — корректировка/период
- `file_name_line1` + `file_name_line2` — имя файла (перенос)
- `ifns_full_name_and_code`, `ifns_code_footer` — подпись ИФНС внизу
- `stamp_*` (3 поля) — для штампа оператора ЭДО в footer

6 статических:
- `form_knd_code`, `title_line*`, `label_*` — часть бланка
