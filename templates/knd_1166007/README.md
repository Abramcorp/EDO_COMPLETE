# templates/knd_1166007 — Извещение о вводе сведений

**Форма КНД 1166007** — извещение о вводе сведений из налоговой декларации в электронной форме без ошибок.

Добавляется в итоговый PDF как страница 6 (см. ADR-003).

## Структура директории

```
knd_1166007/
├── blank.pdf              ← чистый бланк ФНС, 1 страница A4
├── fields.json            ← координатная карта
└── reference/
    └── sample_01_*
```

## Где взять blank.pdf

Аналогично КНД 1166002: nalog.ru или извлечение из эталона (стр. 6 в `Романов_УСН_2025.pdf`).

## Список полей

Из эталона (стр. 6):

| Поле | Источник |
|---|---|
| Реквизиты справа-вверху (представитель/декларант) | `taxpayer` |
| Код налогового органа | `ifns_code` |
| Полное имя налогоплательщика + ИНН | `taxpayer.fio + ", " + taxpayer.inn` |
| Наименование декларации + КНД | константы для УСН |
| Номер корректировки | из DeclarationRequest |
| Период | "34" |
| Год | `tax_period_year` |
| Имя файла | `receipt_data.file_name` |
| Полное наименование налогового органа + код | `ifts_info.name + ", " + ifts_code` |

Форма простая — около 12 полей (меньше чем в КНД 1166002).
